Въпрос 1. Какво е REST (кратко)?
Отговор:
REST (Representational State Transfer - репрезентейшънъл стейт трансфер) е архитектурен стил за изграждане на уеб услуги. Тоест, това е начин да получим или променим дадени ресурси чрез стандартни HTTP заявки. REST API позволява на различни приложения да комуникират помежду си по лесен и стандартизиран начин.

Въпрос 2. Какво е DRF Router?
Отговор:
DRF Router е инструмент в Django REST Framework, който автоматично създава URL адреси за различни операции. Това улеснява конфигурирането на API маршрутите и спестява ръчното им изписване.
